Команда git commit -m "сообщение" простыми словами означает:
"Сохрани изменения с комментарием".
Разберём по частям:
git commit — говорит Git: "Сохрани текущие изменения в истории проекта".
-m — это сокращение от "message" (сообщение).
"сообщение" — твой комментарий к этому сохранению, например: "Добавил кнопку на главную страницу".
Пример:
bash
Копировать код
git commit -m "Исправил баг с авторизацией"
Это как если бы ты сделал снимок текущего состояния кода и подписал его, чтобы потом было понятно, зачем ты это сделал.
Если хочешь, могу объяснить, когда именно нужно делать коммит и как писать хорошие сообщения.
